# US-002: [User Story] Edit and delete expenses
Status: todo
Priority: high
Requirement-Refs: REQ-001
Feature-Refs: FEAT-001
Tags: Functional

## Description
As a user, I want to edit and delete expenses so that I can keep my records accurate.

## Acceptance Criteria
- The user can update the title, amount, date, and category of an existing expense.
- Saving an edited expense updates the stored record and the visible expense list.
- The user can delete an existing expense from the list.
- After deletion, the removed expense no longer appears in lists, filters, or summary totals.
