# US-005: [User Story] Use the app locally without login
Status: todo
Priority: medium
Requirement-Refs: REQ-001
Feature-Refs: FEAT-003
Tags: Functional

## Description
As a user, I want to use the expense tracker without signing in so that I can manage my personal expenses quickly in a local browser app.

## Acceptance Criteria
- The app is usable immediately on load without account creation, login, or identity prompts.
- Expense data is stored locally in the browser for the same user.
- Reloading the page keeps previously saved expenses available in the same browser unless the user deletes them.
