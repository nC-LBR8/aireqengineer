# US-001: [User Story] Add an expense
Status: todo
Priority: high
Requirement-Refs: REQ-001
Feature-Refs: FEAT-001
Tags: Functional

## Description
As a user, I want to add an expense with the required details so that I can record my spending.

## Acceptance Criteria
- The app lets the user add an expense with title, amount, date, and category.
- A saved expense appears in the expense list immediately after it is submitted.
- The app prevents saving an expense when any required field is missing.
- The amount field accepts numeric values greater than zero.
