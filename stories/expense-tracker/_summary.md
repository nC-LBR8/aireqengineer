# Summary

## Use Case
- Browser-based personal expense tracker for a single user without login.

## Requirement
- `REQ-001` - Create a browser app for tracking personal expenses for a single user, including expense entry, editing, deletion, filtering, and monthly summaries, without requiring login.

## Features
- `FEAT-001` - Expense record management
- `FEAT-002` - Expense filtering and monthly summaries
- `FEAT-003` - Local single-user access

## Functional Stories
- `US-001` - Add an expense
- `US-002` - Edit and delete expenses
- `US-003` - Filter expenses by month and category
- `US-004` - View monthly totals and category breakdown
- `US-005` - Use the app locally without login

## Non-Functional Stories
- `US-006` - Maintainability: expense logic separation
