# US-001: [User Story] Start, pause, and reset the timer
Status: todo
Priority: high
Requirement-Refs: REQ-001
Feature-Refs: FEAT-001
Tags: Functional

## Description
As a user, I want to start, pause, and reset the timer so that I can control my current focus session.

## Acceptance Criteria
- The app provides start, pause, and reset controls for the current timer session.
- Selecting start begins counting down from the full session duration, or resumes from the paused remaining time.
- Selecting pause stops the countdown without clearing the remaining time.
- Selecting reset restores the current session to its default full duration and stopped state.
