# US-002: [User Story] View the active session and remaining time
Status: todo
Priority: high
Requirement-Refs: REQ-001
Feature-Refs: FEAT-001
Tags: Functional

## Description
As a user, I want to see whether I am in a work or break session and how much time is left so that I can follow the timer without guessing.

## Acceptance Criteria
- The app shows the current session type as either work or break.
- The app displays the remaining time for the active session in minutes and seconds.
- The displayed remaining time updates while the timer is running and stays unchanged while paused.
