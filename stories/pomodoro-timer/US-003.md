# US-003: [User Story] Complete work and break sessions
Status: todo
Priority: high
Requirement-Refs: REQ-001
Feature-Refs: FEAT-002
Tags: Functional

## Description
As a user, I want the app to complete work and break sessions correctly so that I can follow a Pomodoro cycle in the browser.

## Acceptance Criteria
- When a work session reaches zero, the app marks that work session as complete.
- After a work session completes, the app switches to a break session in a stopped or ready-to-start state.
- When a break session reaches zero, the app marks that break session as complete.
- After a break session completes, the app switches back to a work session in a stopped or ready-to-start state.
- Resetting the timer after an automatic session switch resets the newly active session type rather than the previously completed one.
