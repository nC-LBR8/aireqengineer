# US-001: [User Story] Create daily habits
Status: todo
Priority: high
Requirement-Refs: REQ-001
Feature-Refs: FEAT-001
Tags: Functional

## Description
As a user, I want to define daily habits so that I can track the routines that matter to me.

## Acceptance Criteria
- The app lets the user add a new habit with a name.
- A newly added habit appears in the current habit list immediately after it is saved.
- The app prevents creating a habit with an empty name.
