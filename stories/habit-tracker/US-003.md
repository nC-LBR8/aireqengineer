# US-003: [User Story] View habit streaks
Status: todo
Priority: medium
Requirement-Refs: REQ-001
Feature-Refs: FEAT-002
Tags: Functional

## Description
As a user, I want to see a simple streak overview for each habit so that I can understand my recent consistency.

## Acceptance Criteria
- The app shows a current streak value for each habit based on consecutive completed calendar days ending with today.
- The streak value updates after a habit is marked done for today when that completion extends the streak.
- A habit with no completion for today shows a streak of zero.
