# Summary

## Use Case
- Browser-based daily habit tracker for a single user.

## Requirement
- `REQ-001` - Build a browser-based habit tracker where a single user can define daily habits, mark them as done for the current day, and view a simple streak overview.

## Features
- `FEAT-001` - Habit definition
- `FEAT-002` - Daily tracking and streak overview

## Functional Stories
- `US-001` - Create daily habits
- `US-002` - Mark habits done for today
- `US-003` - View habit streaks

## Non-Functional Stories
- `US-004` - Maintainability: habit tracking modules
