# US-002: [User Story] Mark habits done for today
Status: todo
Priority: high
Requirement-Refs: REQ-001
Feature-Refs: FEAT-002
Tags: Functional

## Description
As a user, I want to mark a habit as done for the current day so that I can record my daily progress.

## Acceptance Criteria
- Each habit can be marked as done for the current calendar day.
- A habit marked as done for today is visibly shown as completed in the current day's view.
- Marking a habit as done for today does not mark it as done for other dates.
- Refreshing the page keeps today's completion state available to the same browser user.
