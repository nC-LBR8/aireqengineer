# US-007: [User Story] Pause, restart, and speed settings
Status: todo
Priority: medium
Requirement-Refs: REQ-001
Tags: Functional

## Description
As a player, I want to pause, restart, and adjust game speed easily.

## Acceptance Criteria
- Pause toggles logic updates while keeping current state visible; resume restarts logic without losing state.
- Restart resets score, snake position/length, and spawns initial food.
- Speed setting offers at least three presets (slow/normal/fast) and updates tick rate immediately.
