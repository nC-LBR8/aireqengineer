# US-009: [CHAR] Usability (including basic accessibility)
Status: todo
Priority: medium
Requirement-Refs: REQ-001
Tags: NFR-Usability

## Description
Controls and UI must be intuitive; keyboard-only operation and basic accessibility support required for web.

## Acceptance Criteria
- Default controls are Arrow keys and WASD with Space for pause; keyboard focus order and visible focus indicators exist for interactive UI elements.
- Contrast ratio for primary game elements and UI meets a reasonable minimum (e.g., text/buttons legible on common backgrounds).
- On-screen touch controls appear on small touch viewports and are sized for finger input (recommended hit target >= 44px).
