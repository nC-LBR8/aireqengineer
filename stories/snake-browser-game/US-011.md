## Description
Deliver a codebase that's modular, testable, and documented to ease future changes.

## Acceptance Criteria
- Core game logic (grid, snake, collision, scoring) is implemented as framework-agnostic modules (e.g., ES modules) separated from rendering and input code.
- Unit tests cover core logic: movement, growth, food spawn, and collision detection (basic coverage of edge cases).
- README documents project structure, how to run locally, how to build/deploy, and how to run tests.
- Build and lint scripts included (npm scripts or equivalent) to run dev server and tests.
