# US-006: [User Story] Keyboard and touch input controls
Status: todo
Priority: medium
Requirement-Refs: REQ-001
Tags: Functional

## Description
As a player, I want to control the snake using keyboard on desktop and on-screen controls on touch devices.

## Acceptance Criteria
- Arrow keys and WASD change direction; simultaneous opposite-direction input is ignored.
- On touch-capable devices, an optional overlay of 4 directional buttons appears (or optional swipe support) and functions equivalently to keyboard input.
- Pause/resume can be triggered by the Space key and a visible UI button.
