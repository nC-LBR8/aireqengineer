# US-010: [CHAR] Compatibility — Browser and viewport support
Status: todo
Priority: medium
Requirement-Refs: REQ-001
Tags: NFR-Compatibility

## Description
Support modern desktop and mobile browsers and degrade gracefully if advanced APIs are missing.

## Acceptance Criteria
- Works correctly in latest two major versions of Chrome, Firefox, Safari, and Edge on desktop.
- Functionally usable on mobile Safari and Chrome (responsive UI and touch controls).
- If Canvas is not available, show a clear message indicating the browser is unsupported (or provide a basic DOM fallback if feasible).
