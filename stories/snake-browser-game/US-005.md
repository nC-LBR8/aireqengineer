# US-005: [User Story] Rendering and responsive UI
Status: todo
Priority: medium
Requirement-Refs: REQ-001
Tags: Functional

## Description
As a player, I want clear visuals of the snake, food, score, and state on desktop and mobile browsers.

## Acceptance Criteria
- Game renders the grid, snake, and food with sufficient visual contrast and scale to be playable at common viewport sizes.
- Score and current speed are visible outside or overlaying the canvas at all times.
- UI is responsive: layout adapts for narrow viewports (mobile) and the canvas scales while keeping grid aspect ratio.
