# US-008: [CHAR] Performance Efficiency — Frame rate and input responsiveness
Status: todo
Priority: high
Requirement-Refs: REQ-001
Tags: NFR-Performance

## Description
Ensure rendering and input feel smooth and responsive for the player.

## Acceptance Criteria
- Rendering runs at up to 60 FPS on a modern desktop browser when viewport >= 1280×720 (no unnecessary blocking during gameplay).
- Input-to-action latency (keyboard to visual move) is less than 50 ms on desktop under normal conditions.
- No dropped logic ticks for normal play at default speed on average hardware; frame drops are handled gracefully (logic remains deterministic even if frames are skipped).
