# US-001: [User Story] Game loop and deterministic ticks
Status: todo
Priority: medium
Requirement-Refs: REQ-001
Tags: Functional

## Description
As a player, I want the game to update state on a fixed tick so gameplay timing is consistent and speed settings are reproducible.

## Acceptance Criteria
- Game state advances on a fixed tick interval configurable (default 10 ticks/sec).
- Tick rate can be changed via the settings UI and takes effect immediately.
- Logic updates (movement, collisions, food consumption) run in the tick step, rendering is decoupled from logic to allow stable timing.
