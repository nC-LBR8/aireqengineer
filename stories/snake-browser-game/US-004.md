# US-004: [User Story] Collision detection and game over behavior
Status: todo
Priority: medium
Requirement-Refs: REQ-001
Tags: Functional

## Description
As a player, I want the game to detect collisions correctly and present a clear game-over state with option to restart.

## Acceptance Criteria
- Collisions with the snake's body are detected and immediately set game state to 'game over' on that tick.
- Collisions with walls either cause game over (default) or wrap if a "wrap" mode is added (explicit mode must be stated).
- On game over the UI shows final score and presents a "Restart" button which resets state to initial.
