# Summary

## Use Case
- Classic Snake browser game.

## Requirement
- `REQ-001` - Erstelle eine Browser-App, die das Spiel Snake abbildet

## Features
- `FEAT-001` - Game loop and timing
- `FEAT-002` - Snake movement and grid logic
- `FEAT-003` - Food spawning and scoring
- `FEAT-004` - Collision detection and game over
- `FEAT-005` - Rendering and UI
- `FEAT-006` - Input handling
- `FEAT-007` - Game controls and settings

## Functional Stories
- `US-001` - Game loop and deterministic ticks
- `US-002` - Snake movement and direction changes
- `US-003` - Food consumption and score updates
- `US-004` - Collision handling and game over state
- `US-005` - Game rendering and HUD
- `US-006` - Keyboard and touch controls
- `US-007` - Pause, restart, and speed settings

## Non-Functional Stories
- `US-008` - Performance efficiency
- `US-009` - Usability
- `US-010` - Compatibility
- `US-011` - Maintainability
